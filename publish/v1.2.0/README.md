## 📖 关于本书

市面上大多数手机摄影教程停留在"教你用某个功能"的入门层面。本书不同——**默认你已经会按快门**，我们直接进入大师级的思维方式：

- 不再教"什么是HDR"，而是教"如何用HDR创造超现实的光影"
- 不再教"怎么开人像模式"，而是教"人像模式的光学局限和破解方法"
- 不再教"三分法构图"，而是教"如何在混乱中构建视觉秩序"
