# 4.3 城市街头：建筑的即兴诗

## 街拍不是"街头摄影"

街头摄影的精髓不是"拍街上的东西"，而是**在无序的城市中识别瞬间的秩序**。街头的建筑提供了这个秩序的框架——墙面是背景、窗框是构图、光影是指引。

## 街头建筑的观察法

### 光影先行法
在街头，先找光，再找建筑，最后等人。
1. 发现一束光斜切过一堵墙，形成鲜明的对角分界线
2. 快速判断：光斑的形状、方向、还有多久会移动
3. 等待一个合适的人（推自行车的、遛狗的、穿鲜艳衣服的）走进光束中
4. 连拍模式

### 色彩狩猎法
预先设定一个"今日主题色"——比如今天只拍黄色：
- 黄色的出租车、黄色的墙、穿黄色衣服的人、黄色的招牌
- 这个限制强迫你用一种新的方式看街道
- 一组同色系街拍放在一起有策展级别的视觉冲击力

### 反射叠影法
城市街头的玻璃表面比任何地方都多：
- 公交站牌玻璃映出对街的建筑
- 咖啡馆的玻璃窗同时映出街景——里面喝咖啡的人 + 外面走过的行人 = 一个画面两个人
- 雨后地面水洼倒映城市天际线——蹲下来，手机接近水面，地面积水变成镜面

## 手机街拍的操作哲学

### 速度优先于完美
街拍不等人。手机的优势就是速度：
- 锁定主摄（1x），不要临时切超广角或长焦（切换延时是致命的）
- 音量键当快门——比触屏快
- 在人行道等红灯时已经提前构图——绿灯亮起的那一刻就是你按下快门的那一刻

### 不打扰的拍摄
- 手机比相机更"不可见"——别人不会觉得一个看手机的人在"拍照"
- 假装在看信息、接电话——你实际上在构图
- 戴耳机——你看起来像一个沉浸在自己世界里的行人，而不是一个"摄影师"

## 街头建筑的三种境界

1. **记录**：建筑长什么样（信息性）
2. **演绎**：建筑在我眼里像什么（个人化）
3. **对话**：建筑和城市中的人在发生什么（叙事性）

追求第三种境界：一张好的街头建筑照片里，建筑不是主角——**建筑中的人才是**。而建筑只是为那个人提供了舞台。

> 🧠 **大师心法**：街头建筑的秘密是"等待"——先找到光影构成的舞台，然后等一个合适的主角走进来。最好的街拍都是"等"出来的，不是"找"出来的。
