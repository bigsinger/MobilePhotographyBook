# 3.3 微观静物：一毫米的奇迹

## 手机微距的独特优势

手机是目前最容易进入微距世界的设备——不需要买微距镜头、不需要三脚架、不需要复杂的焦平面计算。你只需要**把手机凑近**。

现代旗舰手机的主摄最近对焦距离约为8-10cm，超广角甚至能更近（3-5cm）。在这个距离上，日常物品变成了另一个世界。

## 微距拍摄的核心技巧

### 稳定是第一优先级
微距摄影中，即使1mm的抖动也会导致完全失焦：
- **双手夹紧**：双手持机，手肘夹紧胸口/腹部，形成一个"人体三脚架"
- **屏住呼吸**：在按快门前呼气，在呼气末端轻点快门
- **寻找支撑点**：把手或手机靠在桌子、墙壁上
- **用音量键当快门**：比触屏点击更稳定

### 手机微距对焦技巧
- 自动对焦在极近距离经常失效——**手动点击你想要对焦的位置**
- 点击后用手指**上下滑动**小太阳旁的滑块——这是手动对焦（部分手机支持）
- 如果实在对不上：后退2cm再试——你很可能超过了手机的对焦极限

### 光线：微距的光量需求
手机靠近物体时，手机本身会遮挡一部分光线。解决方案：
- 从侧方补光，不是上方
- 用另一部手机屏幕从侧面补光——最佳微距补光方案
- 白纸折叠成U形围在物体周围，只留一个缺口让手机拍照

## 微距的构图法则

微距构图有一个独特挑战：**景深极浅**。
- 手机f/1.6的固定光圈 + 极近距离 = 景深可能只有1-2mm
- 这不是缺陷——这是特点

**应对策略**：
1. **尽量平行于主体平面**：不要让镜头倾斜，让手机与主体保持平行——这样可用的景深最大
2. **焦点在"叙事点"上**：拍花蕊，焦点在花粉上；拍水滴，焦点在水滴的倒影上
3. **接受虚化**：微距的浅景深是一种美学——焦外朦胧不叫"糊了"，叫"氛围"

> 🧠 **大师心法**：微距让你重新发现世界——一片叶子的纹理、一颗水珠的折射，里面都有一个宇宙。
