# 5.3 逆天而行：驾驭极端天气

## 为什么"坏天气"是最好的摄影天气

大多数人在好天气才拍照——所以大多数风光照片看起来差不多。当你在恶劣天气出门，你就自动进入了一个**低竞争高回报**的创作领域。

## 雨

### 雨中拍摄的手机保护
- 防水袋（几元钱的透明手机防水袋）——不影响画质
- 最简单的方案：塑料袋 + 橡皮筋，在镜头位置挖一个孔
- 触摸屏在湿水后可能失灵——用音量键当快门

### 雨天的视觉机会
1. **湿漉漉的表面**：柏油路变成镜面、墙壁颜色变深饱和、树叶反光如漆
2. **雨滴作为前景**：透过布满雨滴的玻璃窗拍外面的街景——雨滴是天然的柔焦滤镜
3. **雨中的人**：打伞的人、奔跑的人、在屋檐下躲雨的人——雨提供了叙事
4. **雨丝**：用1/30s左右的"慢"快门（需第三方App如ProCam），让雨滴拉成丝线

## 雾

雾是**天然的留白工具**。它隐藏了远处的杂乱，只留下近处的主体和朦胧的轮廓。

### 雾中的构图法则
- **必须有前景焦点**：一棵树、一个人、一座塔。没有焦点，雾景就是一片灰白。
- **层次是命**：前景清晰 → 中景朦胧 → 远景消失。这个褪变的过程是雾景的全部魅力。
- **拍黑白**：雾景天然适合黑白——去除了色彩的干扰，只剩下灰度的层次渐变。

### 手机雾景技巧
- 手机可能会把雾"看"得比实际更亮，需要-0.5EV曝光补偿
- 关闭HDR（HDR会"消除"雾的朦胧感，把远景提亮——失去了雾的氛围）

## 雪

### 雪景的曝光陷阱
手机测光系统把"白色"当作"太亮"→ 自动降低曝光 → 雪被拍成灰色。

**解决**：手动+0.7到+1.3EV曝光补偿。拍完放大看雪的细节——雪应该是白色带细微晶体纹理的，而不是灰色泥状。

### 飘雪
用1/60s左右的快门速度（部分手机支持专业模式手动调整），飘落的雪花变成白色短线——画面中的动态元素。

### 雪后
雪停后的第一缕阳光——雪面变成巨大的反光板，光线从各个方向均匀地照亮一切。这时的光线质量接近影棚级别。

## "不好好"用手机

当别人因为天气收起手机时，你把它拿出来。不是因为你的手机更防水——是因为你知道：**独特的光线出现在大多数人不愿意去的时刻**。

> 🧠 **大师心法**：好天气人人都能拍——这恰恰是你不该拍的时候。当别人因为天气放下手机，就是你超越他们的机会。
