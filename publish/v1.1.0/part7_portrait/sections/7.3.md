# 7.3 构图与人物：裁切的心理学

## 裁切不是剪裁——是权力的分配

你把画面切在哪里，决定了观众把注意力放在哪里。裁切是一种视觉权力——你选择让观众看什么、不看什么。

## 人体裁切的黄金法则

**核心原则**：在关节处裁切造成"截肢感"，在骨骼中段裁切自然舒适。

| ✅ 安全的裁切位置 | ❌ 会造成不适的位置 |
|-------------------|---------------------|
| 头顶（留少量空间或略微裁切发顶） | 额头正中间 |
| 大腿中段 | 膝盖 |
| 上臂中段 | 肘关节 |
| 腰部 | 手腕 |
| 小腿中段 | 脚踝 |
| 胸部下方 | 脖子 |

## 景别的情感距离

不同的景别传递不同的心理距离：

| 景别 | 裁切位置 | 心理距离 | 适合表达 |
|------|----------|----------|----------|
| 面部特写 | 额头到下巴 | 极度亲密 | 情绪、凝视、脆弱 |
| 头肩 | 胸部上方 | 亲密 | 表情、个性、对话感 |
| 半身 | 腰部 | 社交距离 | 姿势、手势、交流 |
| 大半身 | 大腿中部 | 公共距离 | 肢体语言、服装、动态 |
| 全身 | 不裁切 | 远距离 | 环境关系、气势、故事 |

**手机景别建议**：
- 特写用2x或3x（避免1x靠近造成的面部变形）
- 半身用1x（距离适中，透视自然）
- 全身用1x（后退至3-5米，保持透视正常）

## 留白的方向性

### 眼神方向的留白
人物看向画面的左侧 → 在左侧留出空间。视线被空间承接，画面呼吸顺畅。**如果视线方向被画框截断，会产生紧张和窒息感——除非你主动要这种效果。**

### 动态方向的留白
人物走向右侧 → 右侧留出空间。运动有"去处"，画面有开放感。

### 打破留白法则
当你想表达被困住、被限制、窒息感时——故意在人物视线或运动方向不留空间。这种"装不下"的感觉正好传达了你想表达的情绪。

## 拍摄角度的权力关系

| 角度 | 效果 | 心理暗示 |
|------|------|----------|
| 眼平 | 自然、平等 | 我和你是一样的 |
| 高角度（俯拍） | 脸小、柔弱 | 俯视的温柔或俯视的优越 |
| 低角度（仰拍） | 高大、强势 | 仰视的崇敬或压迫感 |
| 背影 | 隐匿、意境 | 不打扰的观察、孤独、神秘 |

**手机拍摄角度技巧**：手机轻便，可以轻松放到腰部或举过头顶。利用这个优势——仰拍时把手机翻转（镜头在下方），你能获得比相机更低的角度。

## 画幅选择

| 画幅 | 适合 | 不适合 |
|------|------|--------|
| 16:9 横 | 环境人像、双人、叙事 | 单人半身特写 |
| 4:3 竖 | 单人半身/全身、社交分享 | 需要宽幅的场景 |
| 1:1 方形 | 文艺感、对称构图 | 动态/需要方向感的画面 |

> 🧠 **大师心法**：构图不是"把人物放在哪个位置"——是"你想让观众用什么距离和角度感受这个人"。
