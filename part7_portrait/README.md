# 第七篇·人物篇 ⭐

> 人是最复杂的拍摄对象——不是因为有技术难度，而是因为你需要同时管理光线、构图、姿势、表情、情绪以及你与被摄者之间的人际动态。手机人像的挑战不在硬件，在人心。

本篇是全书重点，10个小节构成完整的人像摄影方法论：从**手机人像的底层光学**到**光线的情绪调度**，从**构图的裁切哲学**到六个专项领域——**站姿导演、坐姿调度、动态捕捉、表情引导、环境叙事、群体统筹**——最后以手机特有的**后期美学**收尾。

## 本篇学习目标

1. 理解手机人像模式的光学局限并学会破解
2. 能独立完成从光线判断到姿势引导的完整人像拍摄流程
3. 掌握引导被摄者自然表达的核心沟通技巧
4. 建立自己的人像审美判断体系## 人物篇结构
