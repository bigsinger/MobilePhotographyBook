# 7.4 站姿导演：身体是一首可以调整的诗

## 为什么站姿是最大的难题

当一个普通人在镜头前站起来时，本能反应是：
1. 双脚并拢（军人站姿）
2. 双手垂放（不知道手放哪）
3. 肩膀紧张（耸肩）
4. 表情僵硬（"我在被拍"的焦虑）

结果：一张"受刑照"——人物像被罚站。

**你的工作不是摄影师，是导演。** 你的第一句话不应该是"笑一个"，而是"放松，我们只是随便走走。"

## 站姿三步改造法

### 第一步：破对称

对称是站姿的死敌。让身体各部位**脱离"左右对齐"**：

- 重心放在**单腿上**——胯部自动倾斜，脊椎自然形成S曲线
- 肩膀**一侧高、一侧低**——靠近镜头一侧的肩膀稍微下沉
- 头部**微侧3-5度**——绝对的正面是证件照

**引导语**："重心放在右腿上，左腿放松弯曲。对，这样自然多了。"

### 第二步：寻找手的位置

"手不知道该放哪"——这是每个被摄者的困惑。你的工作不是告诉被摄者"把手放在X位置"，而是**让手有事情做**。

**手机站姿手部八种方案**：

| 方案 | 操作 | 心理效果 |
|------|------|----------|
| 单手插兜 | 拇指露在外面，其余手指自然弯曲 | 松弛、自信、生活化 |
| 手持道具 | 拿咖啡杯、花束、手机、包 | 手有"任务"就不紧张 |
| 手臂轻抱 | 双手在胸前轻搭，不要用力 | 职业感、知性 |
| 单手撩发 | 手指轻触耳侧头发 | 柔美、随意、瞬间的动态感 |
| 后背自然交握 | 双手在背后交握 | 开怀、坦荡、轻松 |
| 扶靠物体 | 手轻搭在墙壁、栏杆、门框 | 借力的放松——心理上和物理上都"有依靠" |
| 插后裤兜 | 双手拇指插后裤兜 | 酷感、不羁（适合男性/中性穿搭） |
| 一高一低 | 一只手插兜+另一只手自然垂放或拿东西 | 不对称=自然（对称=摆拍） |

**铁律**：手不要紧贴身体。手臂和身体之间保持一个小小的空隙——这个空隙让身体"透了一口气"。

### 第三步：延伸线条

站姿的优雅感来自**身体线条的延伸**而非收缩：

- **腿往前伸**：靠近镜头的腿微微向前，脚尖轻点地——视觉上拉长腿部线条
- **头往上顶**：想象头顶有一根无形的线向上拉动脊柱——不是僵硬挺直，是"长高"的感觉
- **下巴微收+微前伸**：镜子前做这个动作有点奇怪，但镜头里——这是"天鹅颈+小V脸"的秘方

## 不同身型的站姿策略

| 身型特点 | 站姿策略 | 手机拍摄建议 |
|----------|----------|-------------|
| 娇小型 | 低角度仰拍+腿前伸+视线向上 | 手机举到腰部高度微仰 |
| 丰满型 | 45度侧身+手臂和身体间留空隙+一条腿向前 | 用2x从稍远处拍，减少透视变形 |
| 高瘦型 | 正面或微侧+层次穿搭+不必刻意仰拍 | 平视或略低角度 |

## 站姿的心理引导法

不要说"你把腿往前伸"，要说：
- "往前走两步，然后回头看我就好。"
- "你靠着那面墙，当作在等一个老朋友。"
- "假装你在过马路，我刚喊了你的名字——转过来。"

**原则**：给场景，不给指令。让被摄者"进入状态"而不是"摆pose"。

> 🧠 **大师心法**：好的站姿不是摆出来的，是"活"出来的。你的工作不是塑一尊雕像，是创造一个让被摄者忘记自己在被拍的瞬间。
