# 7.7 表情与眼神：拍出灵魂而非皮囊

## 表情是人像的最终得分

焦外糊了但表情绝佳的照片 = 好照片。  
技术完美但表情僵硬的"美"照 = 失败。

你的工作不是把人"拍美"——是**让人在照片中活着**。

## 微笑的引导：不说"笑一个"

### "说茄子"为什么是灾难
"说茄子"调动的口角提肌（假笑肌）。真正自然的笑容调用的是**眼轮匝肌**——眼周肌肉收缩，眼角出现细纹，眼睛微微眯起。

**真笑 = 眼睛在笑。假笑 = 只有嘴在笑。**

### 引发真笑的五种方法

1. **回忆激活**："想一个让你笑出声的人或事情。"——真实的记忆引发真实的微表情。

2. **中间过程抓拍**：不要拍"正在笑"——拍"刚笑完收回来的那一刻"或"快要笑出来的那一刻"。那些过渡表情比巅峰笑容更有故事感。

3. **对话幽默**：一边拍一边闲聊，聊到好笑的事情——不是"你笑一个给我拍"，而是"我们聊天中你自然地笑了"。

4. **不看镜头笑**：看向镜头旁边3-5度微笑——比盯着镜头笑容易放松得多。盯着镜头微笑需要极强的表现力——大多数人的"盯镜头笑"是僵的。

5. **被摄者扮演**："假装你刚在街上遇到了一个老朋友，你很惊喜。"——给一个情境，比给一个指令好一万倍。

## 不笑的表情：情绪光谱

不是每个人像都需要笑容。**放松的平静 = 高级。空白的呆滞 = 失败。**

| 情绪 | 面部特征 | 引导方式 |
|------|----------|----------|
| 平静 | 嘴唇微闭或微启，眼神柔和，面部肌肉放松 | "放松面部，像在发呆一样就行" |
| 沉思 | 眼神看向远处或下方，眉心微微聚拢 | "想一件还没解决的事情" |
| 温柔 | 眼神柔和半眯，嘴角微微上扬（不是笑） | "看那边，想象你最喜欢的人在那" |
| 坚定 | 眼神聚焦，下巴微收，嘴闭合但不紧绷 | "看着镜头，眼神坚定但不凶狠" |
| 脆弱 | 眼神轻微躲闪，头微低 | "闭上眼睛，慢慢睁开，不要看镜头" |

## 眼神的七种方向

| 方向 | 心理效果 | 适用场景 |
|------|----------|----------|
| 直视镜头 | 直接、自信、连接 | 正式肖像、时尚、力量感 |
| 看镜头旁 | 自然、不过分直白 | 日常人像首选 |
| 看远方 | 思考、向往、故事感 | 环境人像、情绪照 |
| 看下方 | 内省、温柔、羞涩 | 女性特写、文艺感 |
| 看上方 | 希望、憧憬、仰望 | 创意风格 |
| 闭眼 | 沉浸、享受、静谧 | 感受阳光/风/音乐的场景 |
| 看镜头"穿过" | 不在看镜头，在看镜头后的世界 | 高级时尚感、距离感 |

### "眼神活化"技巧
呆滞的眼神 = "死鱼眼"。活化方案：
- 让被摄者先闭眼 → 数到三睁开 → 睁开瞬间的眼神最亮最活
- 先看下方 → 听到快门声抬眼看镜头 → 眼动过程中的眼神比固定盯着的眼神有生命
- 让被摄者想一个具体的问题——"今晚吃什么？"——眼神会因为思考而变得活跃

## 面部角度的秘密

| 角度 | 效果 | 适合 |
|------|------|------|
| 正面 | 对称美、力量 | 五官对称的人 |
| 15度微侧 | 打破对称又不失正面感 | 大多数人 |
| 45度半侧 | 五官立体、下颌线清晰 | 所有人——最显瘦的角度 |
| 60度七分面 | 比半侧更侧，轮廓感更强 | 侧脸好看的人 |
| 90度侧脸 | 剪影感、诗意 | 鼻子和下颌轮廓优美的人 |

**手机角度选择铁律**：如果你不确定，45度半侧是"安全角度"。正面考验五官对称，90度考验侧脸轮廓，45度——每个人的下颌线都好看。

> 🧠 **大师心法**：一张好的人像不是"她很好看"——是"她好像在想着什么"。表情的最高境界是让观众觉得你抓住了一个真实的瞬间，而不是一个摆拍的姿势。
