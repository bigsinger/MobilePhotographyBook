# 第五篇·景色篇

> 好风光不等于好照片。手机拍风光的核心不是记录美景本身，而是**用手机的独特能力重新诠释风景**。

本篇从风光摄影的底层思维开始，教会你在正确的时间站在正确的位置，用手机的计算摄影特性捕捉超越肉眼所见的画面。

## 本篇学习目标

1. 掌握风光摄影的预判体系：读天气、算光线、选机位
2. 用手机的夜间模式和长曝光突破肉眼的暗光限制
3. 将雨、雾、雪等"坏天气"转化为创作优势
