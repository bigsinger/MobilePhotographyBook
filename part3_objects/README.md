# 第三篇·实物篇

> 日常之物，以大师之眼凝视，皆可入画。手机的广角微距能力和计算摄影让你比任何相机用户都更接近物体。

本篇教你用手机**翻译质感、构建场景、捕捉微观纹理**。不需要影棚设备——一张白纸、一扇窗户、一部手机，足矣。

## 本篇学习目标

1. 能用日常物品搭建有叙事感的静物场景
2. 掌握手机微距能力拍出令人震撼的细节照片
3. 理解材质与光线的关系，能将触感"翻译"为视觉
