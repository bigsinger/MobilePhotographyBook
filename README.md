# 手机摄影实战指南

> 当前版本: v1.2.0 | 9篇38节 | 171,000+字符 | 大师级深度

## 关于本书

从重新认识你口袋里的手机开始，到光影、构图、实物、建筑、景色、微观——**七篇基础理论**构建完整的手机摄影知识体系。再到**实战篇**的街拍/旅行/美食/夜景/宠物儿童五大高难度场景攻略，最后抵达**创作篇**的风格、系列、边界、伦理、展示——**从工匠到艺术家的完整路径**。

这不仅是一本教程——是把你手中那部手机变成你最重要的创作工具的系统工程。

## 特色

- 🎯 **纯手机**：全篇无单反/微单/换镜头内容，100%以手机为创作工具
- 🧠 **大师心法**：每节末尾一言总结，来自布列松/索尔·雷特/川内伦子/阿勃丝等摄影大师的创作智慧
- 💡 **计算摄影深度**：不只教"怎么拍"，更解释手机AI做了什么——让你夺回审美控制权
- 📋 **每节三件套**：大师案例+常见翻车对照表+刻意练习——完成"学→避坑→练"闭环
- 📸 **38个刻意练习**：从理论到肌肉记忆，系统性训练你的摄影直觉
- ⭐ **人物篇10节全景**：唯一一本用10个独立小节解构手机人像的书（站姿/坐姿/动态/表情/环境/群体/后期）
