# 第九篇·创作篇

> 技术让你成为合格的摄影师。创作让你成为你自己。前八篇讲的是"怎么拍"，这一篇是"拍什么"和"为什么"——从工匠到艺术家需要跨越的那道门。

本篇不讲参数、不讲构图、不讲光线。讲的是：
- 你怎么从成千上万的手机摄影师中辨识出自己的声音？
- 你怎么从单张的好照片进化到一组有力量的系列作品？
- 你怎么在所有人都用手机拍同一类照片的时代，拍出别人没拍过的东西？
- 你怎么让你的作品被看见——不是在亲友群被夸"好看"，而是在更广阔的视野中被认可？

## 创作五课

| 节 | 命题 | 你将要回答的问题 |
|---|------|-----------------|
| 9.1 | 找到你的风格 | 我的照片为什么是我的，而不是任何人的？ |
| 9.2 | 系列作品的力量 | 如何让10张照片讲的比1张大10倍？ |
| 9.3 | 边界突破 | 手机摄影的红线在哪？越过它有什么？ |
| 9.4 | 拍摄伦理 | 你的照片权在哪里结束，别人的人权在哪里开始？ |
| 9.5 | 作品展示与传播 | 从手机相册到被看见，中间隔着什么？ |

> 🧠 **大师心法**：前八篇教你用手机拍照片。这一篇教你用照片拍自己。技术是梯子，创作是你站在梯子上看到的世界。梯子再稳，看不到什么也无意义。
