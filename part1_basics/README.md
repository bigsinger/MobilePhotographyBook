# 第一篇·基础篇

> 多数人用手机拍了几年照片，却从未真正理解手中这台"超级计算机+相机"的工作原理。这一篇帮你完成从"会用"到"理解"的跨越。

本篇不教你按快门，而是带你进入**手机影像系统的底层逻辑**——多摄协同的数学、计算摄影的算法美学、视觉语言的构建法则。这些知识不会直接让你的下一张照片变好，但它们会**永久改变你看世界的方式**。

## 本篇学习目标

1. 理解手机多摄像头系统的光学特性和协同工作机制
2. 掌握计算摄影的核心原理（Deep Fusion、夜间模式、智能HDR），并学会创造性利用其特性
3. 建立个人的视觉语言体系——不只是"拍到"，而是"表达"
