# 第八篇·实战篇

> 知而不行，等于不知。前七篇讲了原理、光线、构图、人物——这一篇把它们全部投进真实场景的熔炉。街头的不可预测、旅行的装备极限、美食的微观世界、夜景的传感器边界、儿童与宠物的毫秒级反应——每个场景自成法则。

本篇的每个小节都是一个**独立作战手册**——从场景理解到参数预设，从常见翻车到补救方案，从基础拍到进阶挑战。

读完本篇，你能够：
- 在0.3秒内完成街头构图和曝光决策
- 用一部手机完成从风景到人文的完整旅行叙事
- 把一碗面拍出米其林质感
- 在城市夜景中突破手机传感器的物理极限
- 拍出宠物和儿童的"决定性瞬间"而非模糊残影

## 实战五训

| 节 | 战场 | 核心挑战 | 需调用前七篇的技能 |
|---|------|----------|-------------------|
| 8.1 | 街头 | 速度·预判·隐匿 | 构图(1.3)+光线(2.1)+环境(7.8) |
| 8.2 | 旅行 | 全场景覆盖·装备轻量化 | 风光(5.1)+建筑(4.1)+人物(全7篇) |
| 8.3 | 美食 | 微观质感·色彩还原·氛围 | 实物(3.1-3.3)+光线(2.2)+后期(7.10) |
| 8.4 | 夜景 | 传感器极限·噪点控制 | 光线(2.1-2.3)+计算摄影(1.2) |
| 8.5 | 宠物/儿童 | 不可预测·高速反应 | 动态捕捉(7.6)+表情(7.7)+构图(7.3) |

> 🧠 **大师心法**：单一技巧在真空中有效，在场景中失效。实战篇教你的是——当十种变量同时发生变化时，如何凭直觉做出最优的三十个微决策。
