# 8.1 街头摄影：在移动中捕捉决定性瞬间

## 街头摄影不是街拍

街拍是看到好看的穿搭抓一张。街头摄影是**在公共空间的流动中，捕捉人与人、人与城市之间偶然发生的、不可复制的视觉关系**。前者关心美，后者关心**真相的瞬间**。

布列松的"决定性瞬间"在手机时代有了新定义：不是等待那个唯一的完美时刻，而是在一秒内完成——看到→判断→构图→曝光→快门——这一串决策。手机让这串决策的执行时间从相机的2-3秒压缩到0.3秒，但前提是你已经**把技术变成肌肉记忆**。

## 街头摄影师的手机预设（出门前必设）

在出门前花10秒做好以下设定——它们会成为你0.3秒决策的基础：

| 设定 | 推荐值 | 原因 |
|------|--------|------|
| 启动方式 | 锁屏左滑或双击电源键 | 从看到到拍摄少按3个键 |
| 默认焦段 | 主摄1x（等效24-28mm） | 最接近人眼视角，无需判断切换 |
| 曝光补偿 | -0.3至-0.7 EV | 街头高光多，欠曝保护高光细节 |
| HDR | 开（自动） | 大光比街道场景必备 |
| 连拍模式 | 已启用 | 决定性瞬间不在第一张 |
| 闪光灯 | 关闭 | 街头摄影禁用闪光 |
| 声音 | 静音 | 尊重、隐匿 |

## 街头构图的"一秒法则"

你不可以在街头花三秒构图——目标已经走出画面了。一秒内完成的构图决策树：

1. **主体识别**（0.1秒）：画面中谁是主角？一个人？两个人的关系？一个人和环境的对比？
2. **快速取景**（0.3秒）：手机举到大概位置——不要追求完美，范围内即可
3. **微调角度**（0.3秒）：上下微调改变透视关系，左右微调改变主体位置
4. **按下快门**（0.3秒）：不是一张，是连拍3-5张

> 一秒法则的本质：不要用取景器找画面，用眼睛找画面。取景器只是把眼睛已经确定的画面框进去。

## 街头的五种决定性瞬间模式

### 1. 巧合型
两个以上不相关元素在画面中产生了意外的视觉关联。例如：广告牌上的人脸和路过行人表情一致、墙上的涂鸦箭头指向刚好经过的人。
- **技巧**：先发现背景中的"有趣元素"（海报/涂鸦/倒影），然后等待合适的人走进画面
- **手机优势**：轻便可以长时间保持同一构图等待

### 2. 光影型
强烈的光线在街道上创造了临时的"舞台"，人物在舞台上短暂亮相然后消失。
- **技巧**：找到光影交界处——光线切割街道形成亮区和暗区的分界线，等人走到分界线上
- **手机设置**：对亮区测光（点击亮区），让暗区变成剪影或半剪影

### 3. 对比型
两个主体之间形成视觉对比：大小、颜色、方向、速度、情绪。
- **常见模式**：静止vs运动、一个人vs一群人、穿亮色的vs全环境暗色
- **手机技巧**：超广角(0.5x)强化大小对比，长焦(2-3x)强化并置

### 4. 反射型
利用玻璃、水面、汽车漆面创造双重现实。
- **最佳介质**：商店橱窗（白天）、雨后地面（傍晚灯映最好）、公交站玻璃（通勤高峰期）
- **手机技巧**：贴近反射面——手机上缘几乎碰到玻璃——最大化反射面积

### 5. 层次型
画面同时包含近景/中景/远景的人物，每层都在做不同的事，形成空间深度。
- **手机技巧**：f/1.8大光圈下所有层次都在景深内——这是手机相对相机的巨大优势

## 街头摄影的伦理与法律

**在拿起手机对陌生人按下快门前，请先完成这个伦理检查清单**：
- 对方是否处于弱势/脆弱状态？（无家可归、哭泣、受伤）→ 不拍
- 对方是否在一个有合理隐私期待的空间？（医院、学校、宗教场所）→ 不拍
- 拍摄是否会加剧对方当下的困境？→ 不拍
- 拍摄后对方发现时，你能否坦然说明你的意图？→ 不能就不拍

**如果在对方注意到时**：
- 微笑、点头。如果对方示意不希望被拍——删除照片，当面展示已删除
- 对方好奇→简单解释："我在拍这条街的光影/这个建筑的角度/这个有趣的场景"

## 街头摄影的"先拍后调"工作流

街头的光线不可控、主体不可控、背景不可控——所以：
1. **原图保留**：永远保留原图，后期另外导出
2. **黑白转换是街头的最强后期**：去掉了色彩的干扰，只用光影讲故事
3. **裁切是你的二次构图**：原图范围拍大一点，后期裁掉干扰元素
4. **加颗粒而非降噪**：街头的噪点不是缺陷，是质感

## 街头实训练习（请按顺序完成）

**练习1：原地观察（30分钟）**
不看手机。选一个街角，只看。预测谁会走进"那道光"。20分钟后开始拍。目标：10张中有5张是预测并等待到的，而不是碰巧拍到的。

**练习2：一秒法则（1小时）**
手机保持锁屏状态在口袋。看到画面→掏出→解锁→拍摄→放回口袋。全程计时。目标：从看到到拍下<2秒。

**练习3：10张序列（遛一次街）**
今天只能拍10张。每张之前必须问自己：这张照片讲什么故事？没有答案就不按快门。目标：每张都有独立的叙事。

> 🧠 **大师心法**：街头不是猎场，是舞台。你不是猎人，是发现了早就存在的戏剧然后悄悄按下暂停键的人。最好的街拍是——观众觉得你恰好路过，但你其实在那里站了半小时。
